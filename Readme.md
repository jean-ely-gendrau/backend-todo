# Backend ToDo

## Commande NPM

- npm init -y
- npm install express dotenv cors mysql2
- npm install --save-dev nodemon ou  npm install -D nodemon

**Pour supprimer un package NPM**

- npm remove nom_du_package

## Commande du terminal pendant la démo

- Voir toutes les commande passé du terminal :
  - history

  ex. 
  ```bash
   2 npm init -y
   3 npm install express dotenv cors mysql2
   4 npm install --sav-dev nodemon 
   5 npm remove nodemon
   6 npm install --save-dev nodemon
   7 npm --help
   8 npm install --help
   9 npm remove nodemon
  10 npm install -D nodemon
  11 npm --version
  12 npm -v
  ```
